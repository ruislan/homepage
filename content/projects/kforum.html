一个简单、现代、时尚、有趣的论坛社区。

KForum 希望帮助各个需要论坛社区的站点，快速的建立一个稳定、可用、低成本的完全属于自己的论坛社区。

版本更新历史

v0.10.0 alpha

2024-01-29

手机访问自适应和更多帖子相关功能

  • [改进] 全面支持手机访问时页面自适应
  • [改进] TipTap 改进操作栏，较窄时自动换行
  • [功能] 增加回到顶部按钮
  • [功能] 帖子可以被收藏
  • [功能] 自己可以看到收藏的帖子
  • [功能] 帖子可以被分享
  • [功能] 被分享的帖子会在浏览时有明显的展示
  • [改进] 一些底层功能的重构 和 Bug 的修订

v0.9.0 alpha

2024-01-15

管理员除了版主的功能，更多的是在整个站点的结构上进行处理，例如分类、菜单等等。版主的功能主要在于内容的处理，例如审批举报、标签的更改、删除、处理帖子的置顶、关闭、编辑、删除等等。

  • [功能] 后台管理员可以设置和取消版主
  • [功能] 可以看到版主列表
  • [改进] 搜索结果页面现在可以过滤和排序
  • [功能] 版主可以编辑、删除话题或者某个回帖
  • [功能] 版主可以置顶、取消置顶、关闭、打开话题
  • [功能] 版主可以处理帖子的举报
  • [功能] 版主可以管理标签、分类、清理无效图片和更新话题热度
  • [改进] 管理员和版主不会被锁定
  • [改进] 用户增加身份标识展示
  • [改进] 一些其他的界面优化和 BUG 修改

v0.8.0 alpha

2023-12-31

重构为逻辑删除功能和细节更新

  • [改进] 发布帖子中只有图片也应该可以发布
  • [改进] 帖子的次要操作按钮现在默认收缩，点击展开
  • [改进] 反馈的图片替换为twiter开源的 emoji SVG图片，同时支持更多的反馈图片，达到 12 个。
  • [改进] 后台增加 menu 设置，可以增加和减少，以及排序，暂不考虑子 menu。
  • [改进] 搜索增加弹出层，用户可以选择在各个tab中进行搜索，并提供 X 用于清空搜索目录
  • [改进] 主题增加热帖（热力指数排列）、最新（最近发布排列）
  • [重构] 完成了模型重构，底层分割更加合理，扩展开发更容易上手
  • [功能] 能够处理定时任务。（使用 Docker 的 cron 方式来处理）
  • [改进] 主题和帖子全部改为逻辑删除，而非物理删除（重要改进）

v0.7.0 alpha

2023-12-09

支持个性化论坛设置和加强发帖相关功能

  • [功能] 管理员可以设置网站的名称和Logo，还有favicon。打造个性论坛。
  • [功能] 发帖可以发链接
  • [功能] 帖子可以被举报，管理员可以在后台管理举报（接受或者忽略）。管理员接受举报将会删除帖子及其相关内容
  • [功能] 现在话题可以被标签，管理员也可以在后台管理标签（Tag）
  • [功能] 增加标签列表
  • [改进] 重做了 Select 组件，Win 和 Mac 的表现一致
  • [改进] 用户中心的回帖列表现在会根据相同话题进行合并展示

v0.6.0 alpha

2023-11-16

主要目标为实现管理员管理功能

  • [功能] 发帖可以支持图片上传和展示了
  • [功能] 管理员现在有了管理界面
  • [功能] 管理员现在可以新建、更新、删除分类
  • [功能] 管理员现在可以封禁和解封用户，封禁后的用户不能登录，也不能做发帖、更改设置等操作
  • [功能] 管理员现在可以清理掉未使用的图片
  • [功能] 管理员现在可以查看用户
  • [功能] 增加了转场页面，切换页面有了一个较好体验。
  • [改进] 使用 MySQL 作为 DB，SQLite 将不再支持
  • [改进] 美化 404 页面
  • [改进] 字数较短的帖子不应该在底部有渐变颜色
  • [改进] 右侧的分类列表在滑动超出视野时保持在视野之内
  • [改进] 讨论或回复在超长文本状况下的展示造成排版混乱
  • [重构] 增加 Upload 概念，统一处理所有上传图片

计划每个月至少更新一次y版本和不定时更新z版本
