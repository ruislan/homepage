一个简单、现代、时尚、有趣的论坛社区。

KForum 有一个中心目标，就是帮助各个需要论坛社区的站点，快速的建立一个稳定、可用、低成本的完全属于自己的论坛社区。后续的所有功能和测试都会围绕这一个目标来实现。

版本历史

v0.8.0 plan

计划中

主要目标为重构为逻辑删除功能和中英双语言支持

  • [改进] 主题和帖子全部改为逻辑删除，而非物理删除，而删除用户，将会实现物理删除。
  • [改进] 主题增加热帖、最新两个选择，包括叠加分类和标签
  • [改进] 搜索增加弹出层，用户可以选择在各个tab中进行搜索，或者提示用户直接按Enter键，并提供 X 用于清空搜索目录
  • [功能] 支持 PostgreSQL，以便能够通过 Vercel 部署
  • [功能] 中英双语言支持

v0.7.0 dev

测试中

主要目标为国际化支持和丰富帖子功能

  • [功能] 管理员可以设置网站的名称和Logo，还有favicon。打造个性论坛。
  • [功能] 发帖可以发链接
  • [功能] 帖子可以被举报，管理后台可以管理举报。接收举报将会删除帖子及其相关内容
  • [功能] 话题增加选择标签的能力，管理后台可以管理标签（Tag）
  • [功能] 增加标签列表
  • [改进] 重做了 Select 组件，在各个平台表现更为统一
  • [改进] 用户中心的回复可以根据相同话题进行合并

v0.6.0 alpha

2023-11-16

主要目标为实现管理员管理功能

  • [功能] 发帖可以支持图片上传和展示了
  • [功能] 管理员现在有了管理界面
  • [功能] 管理员现在可以新建、更新、删除分类
  • [功能] 管理员现在可以封禁和解封用户，封禁后的用户不能登录，也不能做发帖、更改设置等操作
  • [功能] 管理员现在可以清理掉未使用的图片
  • [功能] 管理员现在可以查看用户
  • [功能] 增加了转场页面，切换页面有了一个较好体验。
  • [改进] 使用 MySQL 作为 DB，SQLite 将不再支持
  • [改进] 美化 404 页面
  • [改进] 字数较短的帖子不应该在底部有渐变颜色
  • [改进] 右侧的分类列表在滑动超出视野时保持在视野之内
  • [改进] 讨论或回复在超长文本状况下的展示造成排版混乱
  • [重构] 增加 Upload 概念，统一处理所有上传图片

v0.5.0 alpha

2023-11-02

主要目标为达到基本论坛最基本功能

  • 注册、登陆、登出
  • 用户中心、查看用户信息、讨论和回帖
  • 修改头像、密码、基础信息
  • 回帖、看帖、改贴、删帖
  • 置顶、锁帖、反馈
  • 搜索

版本说明

x.y.z 历史性版本变更和架构调整x，功能性增减加强变更y，计数性变更重构和 BUG 修订变更z，

计划每个月至少更新一次y版本和不定时更新z版本
